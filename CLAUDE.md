## Project Structure
- Base package: `org.example`
- Entry point: `src/main/java/org/example/Main.java`

## Conventions
- Never hardcode credentials — always use environment variables
- Standard layered architecture: controller → service → repository → model


## Project purpose
- Backend for a password manager


## Requirements
- Login Page

- Login/Signup Button, need to be able to verify each login attempt

- Profile Page

- View all current passwords for websites

- add a password to the website

- modify a password for a previous website

- delete a previous website credentials
